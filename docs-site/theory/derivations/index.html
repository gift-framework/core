    1. Weinberg Angle

    \[\sin^2\theta_W = \frac{b_2}{b_3 + \dim(G_2)}\]

    Calculation

    \[\sin^2\theta_W = \frac{21}{77 + 14} = \frac{21}{91}\]

    Simplification

    \[\frac{21}{91} = \frac{3 \times 7}{7 \times 13} = \frac{3}{13}\]

    Result

    \[\sin^2\theta_W = \frac{3}{13} \approx 0.2308\]

    Experimental value: 0.23122 ¬± 0.00003
